# Strategist Memory Seed

## PRD Process
- Chat interface works well for PRD questioning flow
- When commercial intent detected (paid users, App Store, monetization), add Value Proposition Canvas, Business Model Canvas, and Hypothesis Testing sections
- Shape the idea through conversation, not interrogation
- Output: PRD.md + APP_FLOW.md (screen inventory and navigation)

## APP_FLOW.md is Mandatory
- Downstream agents (@architect, @engineer) depend on it for route/page structure
- Include: every screen, user navigation paths, state transitions
